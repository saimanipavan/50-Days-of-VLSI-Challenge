# Week 2 – Dataflow Modeling

## Overview

Week 2 of the **50 Days VLSI Challenge** focuses on designing **Combinational Logic Circuits** using **Dataflow Modeling** in **Verilog HDL**. During this week, fundamental digital circuits were implemented using **continuous assignments (`assign`)**, **operators**, and **conditional (ternary) expressions** to understand an alternative RTL modeling style.

The week concludes with a **Parameterized N-Bit Arithmetic Logic Unit (ALU)**, demonstrating scalable and reusable RTL design using Verilog parameters.

## Weekly Project Highlight

The **Parameterized N-Bit Arithmetic Logic Unit (ALU)** demonstrates reusable RTL design by allowing the data width to be configured using a Verilog parameter. It supports **eight arithmetic and logical operations** selected through a 3-bit opcode, showcasing scalable digital design using **Dataflow Modeling**.
